# 🎨 Dreamweaver Python - Editor Visual Web Completo

Un editor web completo estilo Dreamweaver con funcionalidad WYSIWYG, edición de código con syntax highlighting, gestión de proyectos y más.

![Dreamweaver Python](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8%2B-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## ✨ Características

### 🖥️ Edición Visual
- **Vista WYSIWYG**: Selecciona y edita elementos directamente en la vista previa
- **Sincronización en tiempo real**: Los cambios en código se reflejan instantáneamente
- **Drag & Drop**: Arrastra elementos para reorganizarlos
- **Toggle de visibilidad**: Oculta/muestra elementos temporalmente

### 📝 Editor de Código
- **Syntax Highlighting**: HTML, CSS y JavaScript con CodeMirror
- **Vista dividida**: Código y vista previa lado a lado
- **Auto-completado**: Sugerencias inteligentes mientras escribes
- **Autocierre de etiquetas**: Cierra automáticamente HTML y CSS

### 🗂️ Estructura Jerárquica
- **Árbol tipo carpetas**: Visualiza la estructura completa del documento
- **Iconos por sección**: Cada tipo de elemento tiene su icono identificativo
- **Expansión/colapso**: Navega fácilmente por secciones anidadas
- **Detección automática**: Identifica automáticamente header, hero, main, footer, etc.

### 🎨 Panel de Propiedades
- **Edición visual**: Modifica ID, clases, estilos sin tocar código
- **Color picker**: Selecciona colores visualmente
- **Atributos**: Agrega, edita y elimina atributos personalizados
- **Estilos inline**: Modifica padding, margin, border, etc.

### 📦 Gestión de Proyectos
- **Explorador de archivos**: Navega por tu proyecto
- **Múltiples archivos**: Trabaja con HTML, CSS y JS simultáneamente
- **Autodetección**: Detecta archivos CSS y JS vinculados
- **Guardado**: Exporta archivos modificados

### ⌨️ Atajos de Teclado
| Atajo | Acción |
|-------|--------|
| `Ctrl+S` | Guardar |
| `Ctrl+Z` | Deshacer |
| `Ctrl+Y` | Rehacer |
| `Ctrl+D` | Duplicar elemento |
| `Delete` | Eliminar elemento |
| `Ctrl+Shift+?` | Mostrar ayuda |
